# YouTube Video Downloader

Simple desktop app for downloading YouTube videos built with Python.

## Features
- Download videos in multiple qualities (1080p, 720p, 480p)
- Audio-only option

## Requirements
- Python 3.8+
- yt-dlp

## Setup
```bash
pip install -r requirements.txt
```

## Run
```bash
python main.py
```
